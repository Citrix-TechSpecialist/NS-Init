rtspalgprofilename¶

The name of the RTSPALG Profile.<br/>Minimum length = 1<br/>Maximum length = 127.

rtspidletimeout¶

Idle timeout for the rtsp sessions in seconds.<br/>Default value: 120.

rtspportrange¶

port for the RTSP.

rtsptransportprotocol¶

RTSP ALG Profile transport protocol type.<br/>Default value: TCP<br/>Possible values = TCP, UDP.
