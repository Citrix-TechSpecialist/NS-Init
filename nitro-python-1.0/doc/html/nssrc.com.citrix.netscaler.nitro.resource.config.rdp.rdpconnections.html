rdpconnections module¶

class rdpconnections¶

Bases: nssrc.com.citrix.netscaler.nitro.resource.base.base_resource.base_resource

Configuration for active rdp connections resource.

username¶

User name for which to display connections.<br/>Minimum length = 1.

domain¶

The domain name.

srcip¶

The client IP address.

srcport¶

The client port.<br/>Range 1 - 65535.

destip¶

The Server IP address.

destport¶

The Client IP address.<br/>Range 1 - 65535.

peid¶

Core id of the session owner.

classmethod get(client, name='', option_='')¶

Use this API to fetch all the rdpconnections resources that are configured on netscaler.

classmethod get_args(client, args)¶

Use this API to fetch all the rdpconnections resources that are configured on netscaler. # This uses rdpconnections_args which is a way to provide additional arguments while fetching the resources.

classmethod get_filtered(client, filter_)¶

Use this API to fetch filtered set of rdpconnections resources. filter string should be in JSON format.eg: “port:80,servicetype:HTTP”.

classmethod count(client)¶

Use this API to count the rdpconnections resources configured on NetScaler.

classmethod count_filtered(client, filter_)¶

Use this API to count filtered the set of rdpconnections resources. Filter string should be in JSON format.eg: “port:80,servicetype:HTTP”.
